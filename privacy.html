2.3 Health Data — Injuries (Optional)

If you choose to declare an injury, we collect the injury type and recovery phase. This is entirely optional and only used to adapt your training plan. This is sensitive data under GDPR Art. 9 and is processed only with your explicit consent.

Legal basis: Explicit consent (GDPR Art. 9.2.a)

2.4 Health Data — Menstrual Cycle (Optional, Female Users)

If you enable cycle tracking, we collect: cycle duration, period duration, last period start date, and symptom level. This feature is disabled by default and can only be activated through a deliberate positive act by the user, constituting explicit consent. This data is never shared with any third party.

Legal basis: Explicit consent (GDPR Art. 9.2.a)

4. International Data Transfers

Strava is a US-based company. Data transfers to the United States in the context of OAuth authentication are governed by Standard Contractual Clauses (SCCs) adopted by the European Commission, pursuant to GDPR Art. 46. Supabase hosts all application data in Europe with no cross-border transfers.

6. Security

  • Encryption in transit: All communications are encrypted via TLS 1.2+ (HTTPS)
  • Encryption at rest: Database storage on AWS benefits from encryption at rest
  • Token security: Strava access tokens are stored securely with restricted access
  • Authentication: Passwords are hashed using bcrypt via Supabase Auth — never stored in plain text
  • Least privilege: Each component of the app accesses only the data strictly necessary for its function
